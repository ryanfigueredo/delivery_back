# Fonte Figtree (opcional)

Para usar a fonte **Figtree** no cupom 58mm:

1. Acesse https://fonts.google.com/specimen/Figtree
2. Clique em **Download family** (botão no canto superior direito)
3. Extraia o ZIP e copie para esta pasta:
   - `Figtree-Regular.ttf` (ou `Figtree-Regular.ttf` do pacote)
   - `Figtree-Bold.ttf` (ou `Figtree-Bold.ttf` do pacote)

Se os arquivos tiverem outro nome (ex.: `Figtree-Regular.ttf`), renomeie para:

- `Figtree-Regular.ttf`
- `Figtree-Bold.ttf`

Sem esses arquivos, o cupom usa Helvetica (fonte padrão do PDF).
